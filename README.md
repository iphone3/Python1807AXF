# Python1807爱鲜蜂
## 一、环境说明
- python3.5

## 二、代码获取
```
# SSH方式
git clone git@github.com:iphone3/Python1807AXF.git

# HTTP方式
git clone https://github.com/iphone3/Python1807AXF.git
```

## 三、安装项目依赖
```
pip install requirements.txt
```
> pip是Python3的版本!

## 四、启动项目
```
python manage.py runserver
```
